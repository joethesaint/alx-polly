# ALX Polling App - Complete Development Setup Guide

## Project Overview
A secure polling application with QR code sharing built with Next.js 15, Supabase, and comprehensive security features.

## Technology Stack
- Next.js 15 (App Router)
- TypeScript
- Supabase (Database & Authentication)
- Tailwind CSS + shadcn/ui
- Jest (Testing)
- RBAC (Role-Based Access Control)

## Quick Start

### 1. Environment Setup
Create `.env.local` file:
```
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_service_role_key
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Database Setup
Run the security migration in Supabase SQL Editor:
```sql
-- Copy contents from security-rbac-migration.sql
```

### 4. Development Server
```bash
npm run dev
```

## Common Issues & Solutions

### Issue 1: Module Resolution Errors
**Problem:** Jest can't resolve module paths
**Solution:** Fixed `jest.config.js` with correct `moduleNameMapper`

### Issue 2: Missing UI Components
**Problem:** Build fails due to missing shadcn/ui components
**Solution:** Created alert.tsx and alert-dialog.tsx components

### Issue 3: RBAC Module Missing
**Problem:** Import errors for RBAC functions
**Solution:** Created comprehensive `lib/rbac.ts` with all security functions

### Issue 4: Next.js 15 Async Params
**Problem:** TypeScript errors with page params
**Solution:** Updated params type to `Promise<{ id: string }>`

### Issue 5: Missing Dependencies
**Problem:** Build fails due to missing packages
**Solution:** Installed @radix-ui/react-alert-dialog and isomorphic-dompurify
